# SpaceX Falcon 9 Landing Prediction

This repository contains my data science capstone project focused on analyzing SpaceX Falcon 9 launch data and building a machine learning workflow to help predict first-stage landing outcomes.

## Project Overview

The project follows an end-to-end data science pipeline:

- Data collection using API and web scraping
- Data wrangling and preprocessing
- Exploratory data analysis with visualizations and SQL
- Interactive dashboard development
- Machine learning for launch outcome prediction

## Key Components

Data Collection:

- API-based collection of SpaceX Launch Data.
- Web Scraping for additional mission-related information.

Data Wrangling:

- Data cleaning and transformation.
- Preparation of variables for analysis and modeling.

Exploratory Data Analysis (EDA):

- Visual Exploration of Launch Success trends.
- SQL-based analysis of mission and payload patterns.

Dashboard:

The Dash app provides interactive exploration of launch site performance and payload impact on mission success. 

Machine Learning:

The final notebook focuses on building predictive models for launch outcomes. 

Technologies Used:

* Python
* Pandas
* Numpy
* Matplotlib
* Seaborn
* Plotly
* Dash
* SQL
* Scikit-learn
* Jupyter Notebook

## How to Run:

1. Clone the repository.
   ``` bash
   git clone https://github.com/Flitzhans/SpaceX_DataScience_Capstone.git
   cd SpaceX_DataScience_Capstone
2. Install Dependencies.
   ``` bash
   pip install pandas numpy matplotlib seaborn plotly dash scikit-learn jupyter
3. Open Notebooks. This includes Jupyter Notebook, JupyterLab or Visual Studio Code and open the notebooks in the project folder.
4. Run the Dashboard.
    ``` bash
    python "11.1 - Capstone Introduction and Understanding the Datasets/spacex_dash_app.py"
Then run the URL shown in the Terminal.
